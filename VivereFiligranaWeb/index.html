Domande Frequenti

  • Posso Cambiare la filigrana?

    Si, è possibile scegliere tra due possibili filigrane, il logo oppure la scritta.

  • Posso regolare l'opacità della filigrana?

    Si, è possibile regolare l'opacità della filigrana dall'apposito selettore a tendina in fase di invio del documento.

  • I file vengono conservati?

    No, i file vengono rimossi al termine della giornata (L'orario per il server potrebbe non corrispondere alle nostre 00:00).
